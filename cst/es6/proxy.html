# Proxy

# 简介

Proxy 对象用于定义基本操作的自定义行为（如属性查找、赋值、枚举、函数调用等）。

# 基本用法

new Proxy(target, handle)

参数： target：Proxy绑定的目标，可以是任何类型的对象，包括原生数组，函数，甚至另一个代理。

handle：绑定的目标需要触发的方法。

用法：

var a = {
    b: 111
}
var proxyA = new Proxy(a, {
    get: function(target, key) {
        return target[key];
    },
    set: function(target, key, value) {
        target[key] = value;
    }
})

与defineProperty不同的是，Proxy绑定的是对象，defineProperty绑定的是对象中的属性，并且Proxy在绑定后会返回一个新的对象：

var a = {
    b: 111
}
Object.defineProperty(a, 'b', {
    get: function() {
        console.log("get：", b);
        return b;
    },
    set: function(newVal) {
        b = newVal
        console.log("set:", b);
    }
})
