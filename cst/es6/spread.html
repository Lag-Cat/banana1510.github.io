# 扩展运算符

# 基本用法

es6新增了扩展运算符，就是三个点“...”。

# 对对象使用

理解对象的扩展运算符其实很简单，只要记住一句话就可以：

对象中的扩展运算符(...)用于取出参数对象中的所有可遍历属性，拷贝到当前对象之中

let a = {
    a: "1",
    b: "2"
};

let b = {
    c: "3",
    d: "4"
};

let c = {
    ...a,
    ...b
};
console.log(c);
//{a: "1", b: "2", c: "3", d: "4"}

//如果属性相同的话后面会覆盖前面的
let a = {
    a: "1",
    b: "2"
};

let b = {
    a: "3",
    b: "4"
};

let c = {
    ...a,
    ...b
};
console.log(c);
//{a: "3", b: "4"}

在扩展运算符中可以实现深拷贝:

let a = {
    a: "petter",
    b: "john"
};
let b = {
    ...a
};
console.log(a, b);
b.a = "denny";
console.log(a, b);
//{a: "petter", b: "john"} {a: "petter", b: "john"}
//{a: "petter", b: "john"} {a: "denny", b: "john"}

但是其中也存在问题，它不会遍历属性中的对象：

let a = {
    a: "petter",
    b: {
        name: "john"
    }
};
let b = {
    ...a
};
console.log(a, b);
b.b.name = "denny";
console.log(a, b);
//{a: "petter", b: {name:"john"}} {a: "petter", b: {name:"john"}}
//{a: "petter", b: {name:"denny"}} {a: "petter", b: {name:"denny"}}

# 对数组使用

可以作为参数传入函数

function func(x, y) {
    console.log(x, y);
}

func(...[1, 2]);
//1 2

可以复制数组

let a = [1, 2, 3, 4, 5];
let b = [7, 8, 9, 10];
let c = [...a, ...b];
console.log(c);
//[1, 2, 3, 4, 5, 7, 8, 9, 10]

扩展运算符可以与解构赋值结合起来，用于生成数组

const [first, ...rest] = [1, 2, 3, 4, 5];
first // 1
rest // [2, 3, 4, 5]

需要注意的一点是：

如果将扩展运算符用于数组赋值，只能放在参数的最后一位，否则会报错。

const [...rest, last] = [1, 2, 3, 4, 5];
// 报错
const [first, ...rest, last] = [1, 2, 3, 4, 5];
// 报错

将字符串转为真数组

console.log([..."banana"]);
//["b", "a", "n", "a", "n", "a"]
