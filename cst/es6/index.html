# 前言

本篇主要是本人在学习、复习ECMAScript 6（下文称为“es6”）过程中做的笔记与自己思考后的结果，如果有不对之处请尽情提出，侵删。

# 学习网站（参考网站）

阮一峰：https://es6.ruanyifeng.com/

MDN：https://developer.mozilla.org/zh-CN/docs/Web/JavaScript

# 书籍

《JavaScript高级编程》、《JavaScript权威指南》

# 简介

es6相对于es5新增了许多的特性，提供了许多新的方法以及对象，给开发者提供了更多语法糖，使得在日常学习过程、开发以及维护的工作上能更快速高效得进行。
