# vue和vue-cli的安装

# 简介

本文主要使用npm来安装vue和vue-cli。

# 1. 安装node

后续安装和使用都需要用到node.js，所以需要先安装node。 node.js官网: https://nodejs.org/en/ 安装成功后通过node -v成功显示版本号说明安装已成功

node -v
v12.4.0

# 2. 包管理器

npm在国内使用可能会出现下载缓慢或者下载超时等问题，解决这一问题可以使用 cnpm

npm install -g cnpm -registry=https://registry.npm.taobao.org

或将npm指向的数据源改为淘宝镜像源

npm config set registry https://registry.npm.taobao.org

# 3. 安装 vue-cli

vue-cli提供vue项目生产调试及打包部署的配置。 这里需要全局安装，需要执行下面的命令。

vue-cli2.x安装方式(注意：如果是vue2.x升级vue3.x需要先卸载vue-cli再安装vue/cli):

npm install -g vue-cli

vue-cli3.x安装方式:

npm install -g @vue/cli

安装完成后运行vue -V，能显示版本号的话就表明安装成功了。

vue -V
3.12.0

# 4. 创建vue项目

vue创建项目默认路径为c:\user\你的计算机用户名下面，如果不希望将项目创建在c:\user\你的计算机用户名下的话，需要自己进入指定的路径。这里可以通过右键新建文件夹，进入文件夹，然后shift+鼠标右键，选择在此处打开命令行(win7)或在此处打开powershell(win10)，如果使用vscode，可以直接在vscode的终端中输入命令。

或者可以通过命令创建目录并进入：

mkdir vue-project && cd vue-project

vue-cli2.x和vue-cli3.x中创建项目的命令有所变化

vue-cli2.x:

vue init webpack 项目名

vue-cli3.x

vue create 项目名

操作成功后，会询问你是否需要帮你提前设置好相关的配置，这里可以一直回车按照默认配置安装。
